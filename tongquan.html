Tổng quan về CSS

CSS (Cascading Style Sheets) là ngôn ngữ dùng để tạo kiểu dáng cho trang web: màu sắc, chữ, bố cục, hình ảnh, hiệu ứng... giúp trang web đẹp và chuyên nghiệp hơn.

✔ Ví dụ CSS

h1 {
    color: orange;
    text-align: center;
}
p {
    font-size: 18px;
    color: blue;
}

✔ 3 cách sử dụng CSS

Loại Chức năng
Internal CSS Viết trong thẻ <style>
External CSS Tạo file .css riêng và liên kết bằng <link>
Inline CSS Viết trực tiếp trong thẻ HTML
