Thuộc tính định dạng trong CSS

Thuộc tính định dạng (Formatting) giúp thay đổi màu sắc, kích thước chữ, nền, viền,... làm trang web đẹp mắt hơn.

✔ Các thuộc tính thường dùng

Thuộc tínhMô tả
colorMàu chữ
font-sizeKích thước chữ
background-colorMàu nền
borderĐường viền
text-alignCăn lề chữ (trái, phải, giữa)

✔ Ví dụ minh họa

h1 {
    color: red;
    text-align: center;
}
p {
    font-size: 20px;
    background-color: yellow;
}
